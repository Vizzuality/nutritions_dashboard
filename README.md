# README

## Dependencies

* Ruby 2.3.1
* Rails 5.0.0.1


## Project setup

* Make sure you have the correct ruby version installed
* Install the project gems: `bundle install`
* That is all! 


## Running the project

* Start the rails server `rails server` or `rails s`
* Point your browser to `http://localhost:3000`
